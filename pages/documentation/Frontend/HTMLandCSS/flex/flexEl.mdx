# Размещение флекс элементов в перевернутом ряду в CSS
Значение row-reverse в свойстве flex-direction позволяет выстраивать блоки в ряд, прижимая их к правому краю. При этом происходит переворот порядка - первый блок будет самым правым:

```js showLineNumbers copy 
<div class="parent">
	<div class="child">1</div>
	<div class="child">2</div>
	<div class="child">3</div>
</div>
```

```css showLineNumbers copy
.parent {
	display: flex;
	flex-direction: row-reverse;
	width: 300px;
	height: 200px;
	border: 1px solid red;
}
.child {
	width: 50px;
	height: 50px;
	border: 1px solid green;
}
```
