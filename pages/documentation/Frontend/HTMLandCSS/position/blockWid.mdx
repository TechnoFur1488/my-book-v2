# Блок на всю ширину экрана в CSS

Чтобы фиксированный блок занял всю ширину экрана, ему нужно задать ширину 100%:

```js showLineNumbers copy
<div id="elem"></div>
<div id="content">
	длинный текст, чтобы появилась полоса прокрутки
</div>
```

```css showLineNumbers copy
#elem {
	position: fixed;
	top: 0;
	left: 0;
	width: 100%;
	height: 100px;
	border: 1px solid green;
}

#content {
	margin: 0 auto;
	width: 300px;
	text-align: justify;
	font: 16px Arial;
}
```
