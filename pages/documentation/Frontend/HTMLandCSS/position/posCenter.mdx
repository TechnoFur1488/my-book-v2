# Центрирование через абсолют в CSS

Существует хитрый способ центрирования элемента через абсолютное позиционирование. Можно сделать так, что элемент будет размещен ровно в центре окна браузера. Для этого нужно не задавать размеры элементу и при этом задать позиционирование со всех четырех сторон:

```js showLineNumbers copy
<div id="elem"></div>
```

```css showLineNumbers copy
#elem {
	position: absolute;
	top:    30px;
	right:  30px;
	bottom: 30px;
	left:   30px;
	
	border: 1px solid green;
}
```

## Задания

### Задание 1
С помощью описанного способа сделайте блок, находящийся по центру экрана и отступающий со всех сторон по 100px.

### Задание 2

С помощью описанного способа сделайте блок, отступающий сверху от края экрана 100px, справа - 200px, снизу - 50px, а слева - 10px.
